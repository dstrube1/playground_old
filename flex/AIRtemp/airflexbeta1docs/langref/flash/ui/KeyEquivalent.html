Packageflash.ui
Classpublic class KeyEquivalent
InheritanceKeyEquivalent Object

Representation of key equivalents for menu items and (potentially) other widgets. A KeyEquivalent instance will hold the character code for the base keystroke and a series of boolean values to indicate modifier keys.

See also

flash.events.KeyboardEvent
flash.display.NativeMenuItem
