Packageflash.errors
Classpublic class SQLError
InheritanceSQLError Error Object

Provides detailed information about a failed operation. When an error occurs with a SQL database operation, the SQLConnection or SQLStatement instance dispatches a SQLErrorEvent. Information about the error in the form of a SQLError instance can be accessed from the SQLErrorEvent object's error property.

This class provides properties containing the error code, a text message containing the details of the error, and the operation that caused the error to occur.
