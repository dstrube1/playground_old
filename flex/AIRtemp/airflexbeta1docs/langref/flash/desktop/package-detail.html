PackageClasses
 

The flash.desktop package contains classes used for copy-and-paste and drag-and-drop operations, as well as the Icon class, used to define system icons used by a file. This package is only available to content running in the Adobe Integrated Runtime.


Classes
 ClassDescription
 ClipboardManager Provides access for copying data to and from the system clipboard.
 DragActions Defines string constants for the names of the drag-and-drop actions.
 DragManager Coordinates drag-and-drop operations.
 DragOptions Specifies which drag-and-drop actions are relevant to a drag operation.
 Icon An object that defines the system icons used by a file.
 TransferableData Provides a container for transfering data and objects through the clipboard and through drag-and-drop operations.
 TransferableFormats Defines constants for the names of the standard data formats used with the TransferableData class.
 TransferableTransferMode Defines constants for the modes used as values of the transferMode parameter of the TransferableData.dataForFormat method.
