Packageflash.events
Classpublic class NativeDragEvent
InheritanceNativeDragEvent MouseEvent Event Object

NativeDragEvents are dispatched by the DisplayObjects involved in a drag-and-drop operation.

The initiating object will dispatch:

  • nativeDragStart - when the drag operation begins.
  • nativeDragComplete - when the user releases the dragged item (whether or not the drop was accepted).

The initiating object is the DisplayObject passed to the DragManager in the call to DragManager.doDrag() that began the drag operation.

Potential target DisplayObjects will dispatch:

  • nativeDragEnter - when the drag gesture passes within the object boundary.
  • nativeDragOver - while the drag gesture remains within the object boundary.
  • nativeDragExit - when the drag gesture leaves the object boundary.
  • nativeDragDrop - when the user releases the dragged item over the object and the object has accepted the drop with an earlier call to DragManager.acceptDragDrop().

Typically the handler for the nativeDragEnter or nativeDragOver events will evaluate the data being dragged, along with the drag actions allowed, to determine whether a DisplayObject should accept the drop. To set the DisplayObject as an eligible target, the event handler must call DragManager.acceptDrop()function, passing in a reference to the target DisplayObject. If the user then releases the mouse button, the DisplayObject will become the drop target and dispatch the nativeDragDrop event.

Any DisplayObject can be a drop target.

NativeDragEvent()Constructor
public function NativeDragEvent(type:String, bubbles:Boolean = false, cancelable:Boolean = true, localX:Number, localY:Number, relatedObject:InteractiveObject = null, transferable:TransferableData = null, actionsAllowed:DragOptions = null, dropAction:String = null, ctrlKey:Boolean = false, altKey:Boolean = false, shiftKey:Boolean = false)

Creates an Event object with specific information relevant to native drag-and-drop events. Event objects are passed as parameters to event listeners.

Parameters
type:String — The type of the event. Possible values are: NativeDragEvent.NATIVE_DRAG_ENTER, NativeDragEvent.NATIVE_DRAG_OVER, NativeDragEvent.NATIVE_DRAG_DROP, NativeDragEvent.NATIVE_DRAG_EXIT, NativeDragEvent.NATIVE_DRAG_START, and NativeDragEvent.NATIVE_DRAG_COMPLETE.
 
bubbles:Boolean (default = false) — Determines whether the Event object participates in the bubbling phase of the event flow.
 
cancelable:Boolean (default = true) — Determines whether the Event object can be canceled.
 
localX:Number — The horizontal coordinate at which the event occurred relative to the containing sprite.
 
localY:Number — The vertical coordinate at which the event occurred relative to the containing sprite.
 
relatedObject:InteractiveObject (default = null) — The TransferableData object containing the data to be transfered.
 
transferable:TransferableData (default = null) — The DragOptions object defining the allowed actions (move, copy, or link).
 
actionsAllowed:DragOptions (default = null) — The action set by the drop target. This parameter is used only for the NativeDragEvent.NATIVE_DRAG_COMPLETE event.
 
dropAction:String (default = null) — Indicates whether the Control key is activated. On Macintosh computers, this property represents the Command key.
 
ctrlKey:Boolean (default = false) — Reserved for future use.
 
altKey:Boolean (default = false) — Indicates whether the Shift key is activated.
 
shiftKey:Boolean (default = false)
