Packagemx.effects.effectClasses
Classpublic class WipeLeftInstance
InheritanceWipeLeftInstance MaskEffectInstance EffectInstance EventDispatcher Object

The WipeLeftInstance class implements the instance class for the WipeLeft effect. Flex creates an instance of this class when it plays a WipeLeft effect; you do not create one yourself.
