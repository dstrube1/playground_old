PackageClasses
 
(Adobe Flex Charting 3 components only)

The mx.charts.renderers package contains the series classes for the Flex charting components.


Classes
 ClassDescription
 AreaSeries Defines a data series for an AreaChart control.
 AreaSet A grouping set that can be used to stack AreaSeries objects in any chart.
 BarSeries Defines a data series for a BarChart control.
 BarSet A grouping set that can be used to stack or cluster BarSeries objects in any chart.
 BubbleSeries Defines a data series for a BubbleChart control.
 CandlestickSeries Represents financial data as a series of candlesticks representing the high, low, opening, and closing values of a data series.
 ColumnSeries Defines a data series for a ColumnChart control.
 ColumnSet ColumnSet is a grouping set that can be used to stack or cluster column series in any arbitrary chart.
 HLOCSeries Represents financial data as a series of elements representing the high, low, closing, and, optionally, opening values of a data series.
 LineSeries Defines a data series for a LineChart control.
 PieSeries The PieSeries class defines the data series for a PieChart control.
 PlotSeries Defines a data series for a PlotChart control.
