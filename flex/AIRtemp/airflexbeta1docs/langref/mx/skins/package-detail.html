PackageClasses
 

The mx.skins package contains the classes that define the skinning components.


Classes
 ClassDescription
 Border The Border class is an abstract base class for various classes that draw borders, either rectangular or non-rectangular, around UIComponents.
 ProgrammaticSkin This class is the base class for skin elements which draw themselves programmatically.
 RectangularBorder The RectangularBorder class is an abstract base class for various classes that draw rectangular borders around UIComponents.
