Classpublic class InvalidFilterError
InheritanceInvalidFilterError Error Object

This error is thrown when a filter specified for a target contains invalid characters or is malformed. This error is thrown by the following methods/properties:
  • ILoggerTarget.filters if a filter expression in this listis malformed.
