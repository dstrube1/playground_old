Classpublic class AbstractInvoker
InheritanceAbstractInvoker EventDispatcher Object
SubclassesAbstractOperation, HTTPService

An invoker is an object that actually executes an remote procedure call (RPC). For example, RemoteObject, HTTPService, and WebService objects are invokers.

clearResult()method 
public function clearResult(fireBindingEvent:Boolean = true):void

Sets the result property of the invoker to null. This is useful when the result is a large object that is no longer being used.

Parameters

fireBindingEvent:Boolean (default = true) — fireBindingEvent Set to true if you want anything bound to the result to update. Otherwise, set to false. The default value is true
